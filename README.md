# Responsive Contact Us Form

Uma seção de contato moderna, limpa e totalmente responsiva desenvolvida com HTML5 e CSS3. O projeto apresenta um layout em duas colunas que integra informações de localização/suporte e um formulário de envio de mensagens funcional e intuitivo.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica de dados de contato e elementos de formulário (`<input>`, `<textarea>`).
* CSS3: Estilização de caixas de texto, alinhamentos, bordas arredondadas e efeitos de sombra.
* Flexbox / CSS Grid: Organização do layout dividido em duas seções (informações institucionais à esquerda e formulário à direita).

## Funcionalidades e Técnicas Aplicadas

* Layout em Duas Colunas: Divisão visual clara contendo detalhes de endereço, telefone e e-mail separados por uma linha vertical sutil.
* Formulário Estruturado: Campos limpos para preenchimento de nome, e-mail e uma área de texto expandida para a mensagem.
* Design de inputs minimalista: Caixas de texto com cantos arredondados, fontes suaves e placeholders bem identificados.
* Botão Customizado: Botão de ação "Send Now" destacado com a cor de destaque roxa do projeto e efeitos de interação.
